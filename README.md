# HP Code Evaluation

Up and running:

1. Install ruby version [3.0.3](https://www.youtube.com/watch?v=WE5kOjMEQqc) with your manager of choice
2. `bundle` it
3. `bin/rails s` and browse to [localhost:3000](http://localhost:3000)
